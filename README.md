## How to use

### Dev

1. Clone

```
git clone https://github.com/madiodio/mui-styled-components-typescript-vite.git
```

2. Install dependencies

```
yarn install
```

3. Start

```
yarn start
```

### Build

```
yarn build
```

### Serve the app

```
yarn serve
```
